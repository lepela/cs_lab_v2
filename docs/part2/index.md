# 2부. 흐름 — 자동으로 흐르게, 사람이 끊는다 ⚙️

오후입니다. 오전에 다루던 그 데이터가 **어떻게 들어왔는지** 직접 만듭니다 — 메일에서 적재하고(Lab 4), 사람이 품질을 승인하고(Lab 5), 면접을 확정합니다(Lab 6~7).
