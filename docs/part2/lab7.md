> **이번 랩 완성물**: 지원자 조회 → 두 겹 게이트 → 확인 카드로 전형단계를 변경하는 토픽 + 전용 흐름
> **예상 시간**: 60분 · **완성 신호**: "○○○ 서류합격 처리해줘" → 확인 카드 → 전형단계 변경

**60분 타이머.** 흐름(단건 갱신)을 먼저 만들고, 토픽(게이트·카드·HITL)을 짓습니다.

## 왜 토픽인가

Lab 6의 AI적합도 흐름은 이름만 받아 조회·검증·갱신을 **자급자족**했습니다. 전형단계 변경은 왜 토픽이 필요할까요?

| | Lab 6 AI적합도 흐름 | Lab 7 전형단계 토픽 |
|---|---|---|
| 입력 | 이름 + 적합도 값 | 이름만 (전형단계는 대화로 결정) |
| 확인 단계 | 없음 (결정적 실행) | **카드로 사람 확인 필수** |
| 전형단계 선택 | 해당 없음 | 챗봇과 대화로 결정 |

{: .note }
**3층 분담** — 오케스트레이터(조회·되묻기·AI적합도 전달) / **토픽**(게이트·카드·HITL) / 흐름(ID 단건 갱신).

54. 아래 수식을 붙여넣습니다.

    ```
    {
      type: "AdaptiveCard",
      '$schema': "https://adaptivecards.io/schemas/adaptive-card.json",
      version: "1.5",
      body: [
        { type: "TextBlock", text: "전형단계 변경 확인", weight: "Bolder", size: "Medium", wrap: true },
        { type: "TextBlock", text: "지원자: " & Topic.지원자이름, wrap: true, spacing: "Medium" },
        { type: "TextBlock", text: "현재 전형단계: " & Topic.지원자Record.ReviewStatus.Value, wrap: true },
        { type: "TextBlock", text: "변경 후: " & Topic.전형단계, wrap: true, weight: "Bolder" },
        { type: "TextBlock", text: "위 내용으로 변경할까요?", wrap: true, spacing: "Large" }
      ],
      actions: [
        { type: "Action.Submit", title: "확정", style: "positive",
          data: { actionSubmitId: "review_confirm_v1", decision: "confirm" } },
        { type: "Action.Submit", title: "취소", style: "destructive",
          data: { actionSubmitId: "review_confirm_v1", decision: "cancel" } }
      ]
    }
    ```

    ![적응형 카드 수식 붙여넣기 완성](../../assets/lab7/54.png)


    {: .note }
    출력이 보이지 않는 다면. `스키마 편집`의 값을 아래로 덮어 씁니다.

    ```
    kind: Record
    properties:
        actionSubmitId: String
        decision: String
  ```

    {: .note }
    적응형 카드의 기본 디자인 편집은 json모드로 수행할 수 있습니다. 다만 이때는 레이아웃 편집만 가능하고 변수를 지정하거나 가변적인 수식의 값은 정의가 되지 않습니다. 수식 카드 일때 이 작업이 가능하지만 다시 json모드로 복귀가 되지 않습니다. 레이아웃의 고민이 필요하면 json모드로 먼저 배치를 해보고 수식 모드로 확정하시는걸 추천합니다.

    {: .warning }
    카드 body에 **TextBlock과 FactSet을 섞으면** Power Fx가 스키마를 병합해 `"facts": null` 오류가 발생합니다. TextBlock 단일 타입만 사용하세요.
